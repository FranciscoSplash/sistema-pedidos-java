# 🛒 Sistema de Pedidos em Java

Projeto desenvolvido em Java com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO).

## 📌 Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Criação de pedidos
- Cálculo do total do pedido
- Controle de estoque

## 🧱 Estrutura do Projeto

- `Produto` → Representa um produto (nome, preço, estoque)
- `Cliente` → Dados do cliente
- `Pedido` → Gerencia os pedidos
- `ItemProduto` → Relaciona produto e quantidade
- `Main` → Interface via console

## 💻 Tecnologias

- Java
- Programação Orientada a Objetos (POO)
